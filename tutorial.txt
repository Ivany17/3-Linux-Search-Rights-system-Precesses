env - список всех глобальных перемен
date - дата
cal - календарь
where - найти чт о-то конкретное
nano - текстовый редактор, который не имеет своего графического представления 
CTRL+S - сохранить
CTRL+X - выйти из nano
CTRL+C - завершение процесса
CTRL+Z - остановка процесса
jobs - показывает все остановленные процессы
fg 1 - запускает первый процесс (fg 2 - запускает второй) 
bg 1 - запускает первый процесс в фоновом режиме
kill - останавливает процесс
kill -9 - безусловная остановка процесса
file & - открыть file в фоновом режиме
killall file - закрыть все файлы
exit - выход из консоли
| - pipe (перенаправление между командами)
sort - сортирует файлы
env | sort - отсортированный список всех глобальных перемен
grep XDG - отображает строки, соответствующие шаблону (XDG - шаблон)
--sort=version - сортировка по возрастанию
-r --sort=version - сортировка по убыванию
\ - экран
touch file{1,2,3} - создадутся файлы file1, file2, file3
touch file{1..3} - создадутся файлы в диапазоне (file1, file2, file3)
locate - поиск по файловой системе
man/info/help - manuals
wc - считает строки, слова для каждого файла
ln - ссылка
history - история

/* Система прав */
drwxrwxr-x где
d - первый символ обозначает папку, символ, файл
первые три символа rwx - владелец
вторые три символа rwx - группа владельцев
третие три символа r-x - все остальные
r - read
w - write
x - execute
sudo - включает права супер администратора в Linux
sudo su - открывает приглашение от имени администратора
apropos - посмотреть все команды
apropos ch - все команды на change
chmod - change modification, изменяет права доступа к файлу
chmod u+x file - дает права user execute в file
chmod u-x,g=r,o= file - отбирает права user execute, group может только read, other - ничего  в file
ugoa
u - user
g - group
o - other
a - all
